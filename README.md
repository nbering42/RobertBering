## Photo troubleshooting
All referenced image files are included in /assets. If photos break on GitHub Pages, check:
1. The /assets folder was uploaded.
2. File names match exactly, including capitalization.
3. The site is being served from the repository root.

## Launch
Upload every file and the entire assets folder to the root of your GitHub repository.
