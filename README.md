# Robert Bering Recruiting Platform v7 Flat Upload

This package is designed for GitHub browser upload when folder upload is not working.

## Important
There is NO assets folder in this version. All photos, SVGs, HTML, and CSS files are at the repository root.

## Upload steps
1. Delete the old files from the GitHub repository, including any `assets` folder.
2. Unzip this file on your Mac.
3. Open the unzipped folder.
4. Press Command + A to select everything inside the folder.
5. Drag all selected files into GitHub's Upload files page.
6. Commit changes.

## Why this version exists
GitHub's browser upload can be inconsistent with folders. This flat version avoids that problem by placing every referenced file in the same root directory as the HTML files.
