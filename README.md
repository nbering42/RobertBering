## Upload steps
1. Delete current repo files.
2. Unzip this package.
3. Open the folder.
4. Press Command + A.
5. Drag all files into GitHub Upload files.
6. Commit changes.
7. Wait for GitHub Pages deployment.
8. Hard refresh site with Command + Shift + R.
